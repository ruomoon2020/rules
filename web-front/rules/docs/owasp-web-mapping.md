# OWASP Web 映射（前端）

| 风险 | 前端规则 / 控制 | 验证 |
|---|---|---|
| XSS | 禁止裸 `v-html`，使用安全 HTML 封装 | 安全 review / 单测 |
| 认证与会话 | 不存储密钥；401 恢复不循环 | 集成测试 |
| 访问控制 | 路由 / 按钮仅作 UX，后端鉴权为边界 | 越权联调 |
| 供应链 | 依赖治理、锁文件、许可证与漏洞扫描 | CI / 依赖审查 |
| 敏感数据暴露 | 日志脱敏、下载鉴权、关闭生产 sourcemap | 构建与配置 review |
| 可用性 | 分页、取消过期请求、懒加载、性能预算 | 性能用例 / Web Vitals |

本映射不是替代后端威胁建模；完整接口和权限风险仍以 `web-backend/rules/docs/owasp-api-top10-mapping.md` 与项目安全评审为准。
