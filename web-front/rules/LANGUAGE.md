# 语言约定

| 范围 | 语言 | 说明 |
|---|---|---|
| `shared/*.md` | 简体中文 | SSOT 正文 |
| `codex/*.md`、`README.md` | 简体中文 | 团队主文档 |
| `cursor/*.mdc` 正文 | 简体中文 | 与 shared 一致，便于团队审阅 |
| `cursor/*.mdc` 的 `description` | 中文或简短英文 | 在 Cursor 规则列表中识别即可 |
| `evals/*.md` | 简体中文 | 评测说明 |
| `examples/*` 注释 | 简体中文 | 接入说明 |

维护时：**新增规则优先中文**；若引用代码标识符（`BaseTable`、`pnpm lint`）保持英文原文。
