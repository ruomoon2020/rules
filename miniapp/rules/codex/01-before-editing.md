# 01 Before Editing

改代码前先确认：

1. 目标端是否为小程序，是否只需微信小程序能力。
2. 改动属于页面、App 运行时、API/网络、登录态、分包、隐私授权、平台能力、支付分享订阅、性能、发版环境中的哪类。
3. 是否涉及 OpenAPI / schema / generated 类型。
4. 是否会增加主包体积或引入新依赖。
5. 是否调用敏感能力，是否已有隐私用途说明。
6. 是否影响登录态、token、手机号绑定、退出登录。
7. 是否需要同步 `pages.json`、`manifest.json` 或分包配置。
8. 是否改动 `App.vue`、全局错误、web-view、请求域名或环境配置。

不得在未确认规则路由时直接改代码。
