# 12 List Form Pagination

## 列表

1. 列表必须包含 loading、empty、error、content、finished 状态。
2. 下拉刷新重置页码和数据；上拉加载只追加下一页。
3. 筛选条件变化必须重置页码。
4. 删除最后一条后要回退页码或刷新当前页。
5. 禁止重复触发分页请求。

## 表单

1. 字段来自 schema / generated 类型。
2. 必填、长度、格式、枚举、金额、时间必须和后端校验一致。
3. 提交按钮必须防重复点击。
4. 提交失败保留用户输入；成功后按流程返回、刷新或跳转。

## 移动交互

1. 触控区域要足够大，关键按钮不能贴边。
2. 底部操作区要考虑安全区。
3. 弹窗必须可关闭，授权类弹窗不得阻塞所有路径。
