# 13 Upload Download Media

## 上传

1. 上传必须走统一封装，包含鉴权、大小限制、类型白名单和错误处理。
2. 图片上传前应压缩或限制尺寸。
3. 文件上传必须处理进度、取消、失败重试和超时。
4. 上传结果必须以后端返回的 fileId / url 为准，禁止信任本地临时路径。

## 下载

1. 下载文件必须鉴权。
2. 下载后的打开、保存、分享必须处理权限失败。
3. 临时文件要有清理策略。

## 媒体与隐私

1. 相机、相册、文件选择必须先过隐私用途检查。
2. 不得上传 EXIF、定位、身份证等敏感信息，除非业务明确需要且已声明。
3. 图片、视频、音频大小和格式要有上限。
