# 微信小程序合规与审核 Checklist

> 非法律意见；以微信平台最新规范与法务评审为准。编码拒 PR 约束见 `shared/09`、`shared/14`、`shared/21`、`shared/20`；本文件用于发版/法务勾选。

## 隐私与个人信息

- [ ] 采集范围最小必要；隐私政策与实际能力一致
- [ ] 手机号、位置、相机、相册等须用户触发 + 用途说明
- [ ] 拒绝授权有降级，不强制循环弹窗
- [ ] 埋点 / 日志不含 token、手机号、身份证、详细地址

## 分享与订阅

- [ ] 无诱导分享、诱导订阅
- [ ] 分享 path / scene 白名单；无敏感参数
- [ ] 订阅消息由用户动作触发；处理拒绝与模板失效

## 支付

- [ ] 支付参数来自后端；结果以后端订单为准
- [ ] 虚拟支付、类目限制符合平台要求
- [ ] 金额不用浮点计算展示

## 内容与类目

- [ ] 小程序类目与实际业务一致
- [ ] UGC、直播、医疗等高风险类目有专项评审
- [ ] `web-view` 域名白名单；不向 H5 透传 token（编码约束见 `shared/21`）
- [ ] 富文本/公告/评论经消毒或白名单（`shared/23`）

## 技术与安全

- [ ] 密钥、AppSecret 不进仓库；环境分离（`shared/19`：体验/审核版禁连生产 API 与支付）
- [ ] 生产无 mock、调试面板、测试账号
- [ ] 第三方 SDK 有隐私评估记录

## 发版前

- [ ] `build:mp-weixin` 通过；主包体积在预算内
- [ ] 体验版自测：登录、授权、支付、分享、分包跳转
- [ ] 见 `shared/19-release-ops.md`
