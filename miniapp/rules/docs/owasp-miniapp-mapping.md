# OWASP 对照（小程序裁剪）

> 非完整合规认证；用于 PR 与威胁建模自检。编码约束以 `shared/` 为准。

| 风险 | OWASP 参考 | 本规则包 |
|---|---|---|
| 非授权访问 | API Top10 BOLA/BFLA | `06` 登录态；后端鉴权不单靠 UI 隐藏 |
| 伪造请求/重放 | API Top10 | `05` 幂等键；`14` 支付以后端为准 |
| 敏感数据泄露 | MASVS Storage | `06`/`08` 禁止明文 token；`15` 日志脱敏 |
| 注入 | XSS | `23` 富文本/UGC 消毒 |
| 错误配置 | 环境混用 | `19`、`00` §33 体验/审核禁连生产 |
| 供应链 | 依赖漏洞 | `25` audit + 锁文件 |
| 出站安全 | SSRF/开放重定向 | `21` 域名白名单；禁止动态 host |
| 客户端完整性 | 调试入口 | `00` 禁止生产 mock/console |
| 隐私 | 过度采集 | `09` 最小必要 + 用途说明 |

发版前可勾选：Security evals M06–M07、M30–M34 + Resilience M35–M38。
