# Data Privacy & Lifecycle

与 `06-security-authz.md`（鉴权）、`24-data-access-cache.md`（脱敏）互补，覆盖数据分级与生命周期。

## 敏感字段分类

| 级别 | 示例 | 要求 |
|---|---|---|
| 凭证 | 密码、Token、密钥 | 禁止落库明文；禁止日志/缓存 |
| PII | 手机、邮箱、证件、地址 | 脱敏展示；导出受权限控制 |
| 财务 | 金额、账户、流水 | 审计 + 权限码；导出水印（若项目要求） |
| 权限数据 | 角色、数据范围 | 变更须审计（`27-audit-log.md`） |

新字段须在 OpenAPI / 数据字典标注敏感级别。

## 保留与删除

1. 业务数据须定义**保留周期**（如订单 7 年、日志 180 天）。
2. **软删除**：逻辑删除字段 + 查询默认过滤；须说明何时物理清理。
3. **硬删除**：仅合规允许场景；须审计且不可逆提示。
4. 用户注销 / 被遗忘权：按法规做匿名化或删除，禁止仅停用账号仍保留可识别 PII 于对外接口。

## 导出与测试

1. 导出敏感数据：鉴权、审计、条数上限；可选水印 / 下载有效期（见 `14`）。
2. **测试 / 预发**禁止使用生产库；测试数据须脱敏或使用 fixture（见 `15-testing.md`）。
3. 禁止将生产数据 dump 到开发机无加密存储。

## 衍生存储中的生命周期

| 载体 | 要求 |
|---|---|
| 日志 | 敏感字段脱敏；保留周期与归档 |
| 缓存 | 禁止缓存未脱敏 PII；TTL 与失效策略见 `24` |
| MQ | 消息体最小化；禁止长期队列堆积明文凭证 |
| 备份 | 加密、访问控制、保留周期与销毁流程 |

## 禁止

- 在错误响应、Swagger 示例、单元测试断言中写入真实生产 PII。
- 缓存 key 含明文手机号等可枚举敏感标识（可用 hash + 盐，按项目约定）。
