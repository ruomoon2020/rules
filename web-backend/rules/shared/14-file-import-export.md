# File Import / Export

与前端 `14-upload-import-export.md` 对齐。

## 上传

1. 校验类型、大小、扩展名；存储走 OSS/SDK，禁止页面硬编码 AK/SK。
2. 病毒扫描、内容检测按项目要求。

## 导入

1. 预校验：表头、必填列、枚举、行数上限。
2. 异步任务 + 进度；**幂等键**（`taskId` / 文件 hash + 业务键）见 `18`。
3. 高风险导入（权限覆盖等）：影响摘要 + 二次确认。
4. 失败：**行列级错误明细**；禁止仅返回「导入失败」。
5. Excel/CSV 防公式注入；导出脱敏。

## 导出与下载

1. 大数据异步导出；禁止一次性加载百万行到内存。
2. 下载链接：**鉴权 + 有效期** 或一次性 token；禁止永久公开 URL 存敏感数据。
3. 审计：操作人、时间、条件摘要、文件名、结果。

## SQL

批量入库使用批量插入；禁止循环单条 insert（性能）。
